Bake cookies

这题从名字可以看出是有关于COOKIE的,从标题看到JWT,JSON Web Token

进去界面猜测账号密码，使用guest/guest登录成功了…弱口令,进去后抓包看COOKIE。

明显是JSON Web Token,访问https://jwt.io去解密

修改为
{
“name”: “admin”,
“admin”: “true”
}
后果COOKIE后提交发现报错信息貌似返回了密钥

填写上密钥重新生成下提交
